1. Start the server
   Open terminal
   Go to the server folder
   Run: node server.js
2. Open the login page
   Open the client folder
   Open index.html in a browser
3. Login
   Enter your email
   Click Login
   You will be redirected to the dashboard
4. Subscribe to stocks
   Select a stock ticker from the dropdown
   Click Subscribe
   The stock will appear in your list
5. View live updates
   Stock prices update every second
   No page refresh is required
6. Test multiple users
   Open index.html in another browser
   Login with a different email
   Subscribe to different stocks
   Each dashboard updates independently
